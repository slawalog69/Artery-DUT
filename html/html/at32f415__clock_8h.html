void system_clock_config ( void  )

system clock config program

Заметки
the system clock is configured as follow:
  • system clock = hext / 2 * pll_mult
  • system clock source = pll (hext)
  • hext = 12000000
  • sclk = 144000000
  • ahbdiv = 1
  • ahbclk = 144000000
  • apb1div = 2
  • apb1clk = 72000000
  • apb2div = 2
  • apb2clk = 72000000
  • pll_mult = 24
  • flash_wtcyc = 4 cycle
Аргументы
none
Возвращаемые значения
none
